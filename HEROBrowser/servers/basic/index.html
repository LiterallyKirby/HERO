Welcome to the HERO Protocol!

What is HERO?

HERO is a lightweight UDP-based protocol designed for fast, efficient communication.

Features

  • UDP-based for low latency
  • Built-in acknowledgment system
  • Support for encrypted connections
  • Simple packet structure
  • Cross-platform C++ implementation

Protocol Flags

HERO uses five main packet types:

  • CONN (0) - Establish connection
  • GIVE (1) - Send data
  • TAKE (2) - Request resources
  • SEEN (3) - Acknowledge receipt
  • STOP (4) - Close connection

Getting Started

To create your own HERO server:

    #include "HERO.h"
    
    HERO::HeroWebServer server(8080);
    while (server.isRunning()) {
        server.serve();
    }
